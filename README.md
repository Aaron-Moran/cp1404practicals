## Clean code lessons
- Apply the Single Responsibility Principle and keep functions small and focused.
- Use clear, consistent naming and constants in ALL_CAPS for readability.
- Follow the [CP1404 Style Guide](https://github.com/CP1404/Starter/wiki/Style-Guide) for formatting, naming, and structure.
